# Схема полного сумматора для изучения принципов работы алгебры логики в школе

Выполнено на транзисторах 2N2222 и резисторах по 1k и 10k. Схемы можно подключать последовательно. А так же, для удобства рекомендуется вывести элементы управления и индикации.
